# Plan: HU-001 — Inicialización del proyecto

## Enfoque Técnico
1. **Control de versiones (Git):** Inicializar el repositorio en la carpeta raíz y crear el archivo `.gitignore` para no subir archivos pesados o secretos.
2. **Backend:** Crear la carpeta `backend`, inicializar un proyecto Node.js, instalar las librerías necesarias (`express`, `cors`, `dotenv`) y crear un servidor básico (`server.js`) que corra en el puerto 3000.
3. **Frontend:** Crear la carpeta `frontend` utilizando Vite (con React y JavaScript). Instalar y configurar Tailwind CSS para los estilos.
